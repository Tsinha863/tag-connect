# **App Name**: TAG Connect

## Core Features:

- User Role-Based Authentication: Secure login/signup for Students/Workers, Companies, and Admins via Firebase Authentication with role-based access control.
- Student/Worker Profile & CV Management: Students and workers can create/update comprehensive profiles, list skills, job preferences, and upload their CVs (stored in Firebase Storage).
- Company Profile & Job Posting: Companies can create/update their profiles, post new job listings, and manage their existing job advertisements.
- AI-Powered Job Description Tool: Companies can use an AI tool to generate optimized job descriptions, suggesting relevant skills and keywords for better matching and SEO.
- Job Search, Filtering & Application: Students can search and filter job postings by various criteria (e.g., location, skills, job type) and apply to suitable roles.
- Application & Candidate Management: Companies can view and manage applications for their jobs, including shortlisting candidates, viewing CVs, and marking candidates as hired.
- Admin Portal for Platform Management: An admin dashboard allowing TAG MEDIA to approve companies, verify student profiles, disable accounts, and access overall platform metrics.

## Style Guidelines:

- Primary action color: A professional and clean blue (#256BDC) to convey trust and efficiency in line with the staffing industry.
- Background color: A very light, desaturated blue (#EEF5FF) providing a clean canvas that complements the primary color and supports readability.
- Accent color: A vibrant, analogous cyan-blue (#26B2CB) to highlight key information, interactive elements, and create visual interest while maintaining professionalism.
- Headline and body font: 'Inter' (sans-serif) for its modern, highly legible, and objective aesthetic, suitable for a professional platform requiring clarity and excellent readability across devices.
- Use clear, minimal, and intuitive vector icons that support professional visual communication without clutter, ensuring universal understanding.
- Adopt a mobile-first responsive layout with generous white space to enhance readability and user focus, ensuring a clean and uncluttered interface across all screen sizes. Prioritize clear headings and semantic HTML for SEO benefits.
- Incorporate subtle and quick animations for user feedback and transitions, minimizing load times and contributing to a smooth, efficient user experience, aligned with performance requirements.
